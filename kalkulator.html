    Pilih Metode

    Pilih salah satu metode numerik di menu untuk memulai perhitungan.

    Gauss-Jordan RREF
    Selesaikan sistem persamaan linear Ax = b menggunakan eliminasi Gauss-Jordan (Reduced Row Echelon Form).
    INPUT
    Matriks Koefisien [A]
    =
    Konstanta [b]
    * Masukkan koefisien matriks A (kiri) dan konstanta b (kanan) secara terpisah
    Invers Matriks A⁻¹
    Selesaikan sistem persamaan linear Ax = b menggunakan metode invers matriks: hitung A⁻¹ lebih dulu (via [A | I] → [I | A⁻¹]), lalu x = A⁻¹ · b.
    INPUT
    * Kolom kiri (a₁, a₂, a₃, …) = koefisien soal, kolom "x" = penanda variabel baris, kolom "b" = konstanta ruas kanan
    Regula Falsi ROOT FINDING
    Cari akar persamaan f(x)=0 pada interval [a,b]. Metode posisi palsu — gabungan biseksi dan interpolasi linear.
    INPUT
    Iterasi dihentikan berdasarkan jumlah iterasi sistem (tanpa input)
    Newton-Raphson DERIVATIVE
    Cari akar persamaan menggunakan metode iterasi turunan dengan konvergensi kuadratik. Rumus: xₙ₊₁ = xₙ − f(xₙ)/f′(xₙ)
    INPUT
    Interpolasi Lagrange POLYNOMIAL
    Hitung nilai interpolasi P(x) menggunakan polinom Lagrange dari sekumpulan titik data (xᵢ, yᵢ).
    INPUT
    Simpson 1/3 INTEGRATION
    Hitung nilai integral ∫f(x)dx dari a ke b menggunakan aturan Simpson 1/3 komposit. Rumus: I = (h/3)[f(x₀) + 4f(x₁) + 2f(x₂) + … + f(xₙ)]
    INPUT
    * Jika n ganjil akan otomatis dinaikkan 1
    Metode Euler ODE SOLVER
    Selesaikan persamaan diferensial biasa y′ = f(x,y) secara numerik. Rumus: yₙ₊₁ = yₙ + h · f(xₙ, yₙ).
    INPUT
    * Jumlah langkah dihitung otomatis: n = (xₙ − x₀) / h
    HISTORI
    0
    🕒 Belum ada perhitungan
